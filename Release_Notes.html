Purpose

STM32WL-SubGHz-Phy-Rx-Hop example demonstrates demonstrates a transmitter device (implemented on a stand alone Nucleo-WL55JC MB1389 board) and a receiver (implemented on a second Nucleo-WL55JC MB1389 board) that can be configured as a standalone demodulator or a frequency hopping scanner and demodulator.

STM32WL-SubGHz-Phy-Rx-Hop example is being delivered as-is, not necessarily supported by ST.

This example includes the HAL and LL drivers provided in the STM32CubeWL_v1.2.0 package. For further information about these components, see official package.

Update History

Main Changes

Initial release of STM32WL-SubGHz-Phy-Rx-Hop Example

In the STM32WL-SubGHz-Phy-Rx-Hop example, the project is generated with the STM32CubeMX tool to initialize the system, peripherals and middleware stacks. User can open the provided ioc file in STM32CubeMX to modify the settings and add additional peripherals, to customize his final application, or generate project for different compiler/toolchain; i.e. MDK-ARM
